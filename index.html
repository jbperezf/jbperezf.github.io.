HYD Generator

  1. Enter your project name
  2. Select the calculation type
    (Only Channel Analysis is available now. The ability to add different analysis will be comming soon. Please consider donating!)
  3. Download the CSV template and populate the file with the data you want to create you HydraulicToolbox file with
  4. Upload the CSV file with your channel data
  5. Click "Generate .hyd File" to create your HydraulicToolbox file

Report Converter

  1. Upload a HydraulicToolbox report file (.docx)
  2. Click "Convert to CSV" to generate a CSV file
